# NoSQL-Pass-Extractor
Is a Python script designed to extract passwords from websites vulnerable to NoSQL injection.
- This script was originally developed for use in the TryHackMe room:
👉 NoSQL Injection Tutorial [nosqlinjectiontutorial](https://tryhackme.com/room/nosqlinjectiontutorial)
